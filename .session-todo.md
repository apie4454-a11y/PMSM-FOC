- **Setup:**

### 2. Sine-Triangle Modulation @ 51V

  - Set Vdc = 51V
  - Apply same reference: Vmag = 25.54V
  
- **Expected Results (SATURATION):**
  - Phase voltages clip at ±25.5V (insufficient!)
  - Voltage circle constraint: √(Vd² + Vq²) > 25.5V ✗ (exceeds limit)
  - Visible clipping on phase voltage waveforms
  - Current loop loses authority (cannot maintain iq = 1A)

- **Metrics to Log:**

  - Back-calculated V_max = 51/2 (should equal 25.5V)
  - Evidence of saturation in voltage output
  - Current tracking error (iq_cmd vs iq_actual)
